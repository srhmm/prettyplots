\subsection{Line} 
\hypertarget{prettyline}{}

\index{Plot Types!line} 
\index{Plot Types!line!basic}

For basic line plots, you can pass \texttt{pretty line} to the \texttt{axis} keys.

\index{Plot Types!line!with marks}

The default color scheme \texttt{pr-colors}, you can find others in \sectref{sec:style}. For example, 
\begin{figure}[h!] 
	\begin{minipage}{0.65\textwidth} 
		\begin{tcolorbox}
			\begin{minted}{TeX} 
\begin{axis}[pretty line, cycle list name=pr-colors]
    \pgfplotsinvokeforeach{1,...,3}{
    \addplot+ table[x=x, y=y#1] {data/line.tsv}; 
} 
\end{axis}
			\end{minted}
		\end{tcolorbox}
	\end{minipage}
	\begin{minipage}{0.3\textwidth}  
		\begin{tikzpicture}
			\begin{axis}[
				pretty line, ymin=0, ymax=10,
				cycle list name=pr-white-marks, width=\textwidth,  
				]
				%\addlines{data/line.tsv} 
				\pgfplotsinvokeforeach{1,...,3}{
					\addplot+ table[x=x, y=y#1] 
					{data/line.tsv}; 
				}
			\end{axis} 
		\end{tikzpicture}
		%\lines{data/line.tsv}[pretty grid, cycle list name=pr-marks]
		\hfill	
	\end{minipage} 	 
\end{figure}
